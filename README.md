# Welkom in mijn functional programming project. üî±
In deze repo vind je een prototype applicatie voor een interactieve board. Deze applicatie kan het volgende:
* Een statische website maken
* JSON en Query data ophalen
* Data filteren en manipuleren
* Lengte formaten opschonen

# Installatie üõ†
1. Clone deze repo 

**Installeer:**

node modules

` npm install -dev `

express

` npm install express`

nodemon 

` npm install nodemon `

**Bewerking**

package.json
```
 "scripts": {
    "start": "node index",
    "dev": "nodemon index"
  },
```

2. Applicatie starten

` npm run dev `
